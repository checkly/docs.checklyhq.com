## Run Locally

```bash
$ npm install
$ npm start

$ npm run start:api
```

## Build & Deploy

```bash
$ npm run build
$ npm run deploy
```

> Push to any branch will generate deployment preview via Vercel

> Push to `master` will automatically deploy to production via Vercel
